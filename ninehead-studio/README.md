# NineHead Studio — Official Website

Official website for **NineHead Studio**, a mobile game and app development studio and publisher.

## 📁 Project Structure

```
ninehead-studio/
├── index.html           # Home page
├── privacy-policy.html  # Privacy Policy page
├── css/
│   └── style.css        # All styles (design system, components, pages)
├── js/
│   └── main.js          # Animations, cursor, scroll reveal, mobile nav
└── README.md
```

## 🚀 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero, Services, About, Publisher, CTA |
| Privacy Policy | `privacy-policy.html` | Full privacy policy document |

## 🌐 GitHub Pages Deployment

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://<username>.github.io/<repo-name>/`
